# task-manager-api
nodejs task manager api with mongo db

## Installation

```sh
npm i 
```

## Setup
- Setup MongoBD
- config/dev.env
```sh
PORT=3000
MONGODB_URL={UrlToMongoDB}
JWT_SECRET={whatevernameisok}
```

## Start

```sh
npm start
```

## Tests

```sh
npm test
```
